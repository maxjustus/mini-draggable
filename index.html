mini-sortable

Drag-to-reorder for lists, grids, and variable-height layouts. Works with any CSS layout. Touch support included.

GitHub

Basic list

Bind a data array. The plugin auto-splices on reorder.

Drag handles

Disabled items

Pinned items cannot be dragged or swapped into.

CSS grid with varied sizes

Variable height items

No special configuration. Heights handled by the FLIP animation.

Cross-container transfer

Drag items between grouped containers. Arrays auto-splice on transfer.
